# Attracteur de Lorenz
 Des fichiers pour construire une version 3D d'un attracteur de Lorenz. Un programme Python calcule un attracteur de Lorenz, 
calcule l'intersection de la courbe avec un nombre de plans régulièrement espacés. 
Le programme crée un fichier pour chaque plan d'intersection, avec la position des trous 'répertoire svg). Le tout au format SVG, prêt pour 
la découpe laser. Il crée aussi des fichiers images, où chaque plan contient les numéros des points d'intersection, dans l'ordre de la courbe (répertoire numerotage). 

Autres répertoires: 
doc : Un fichier d'explication en tex, et un sous-répertoire avec des photos. 
Rhino : l'attracteur pour l'afficher dans Rhion (modéliseur 3D)
sources : Les sources python
Vision3D : des images.
